the awesomizer.

a library engagement project.

what is it?

The Awesomizer is a library engagment project implemented by the student developers and interns of the University of Michigan - Ann Arbor's Shapiro Design Lab. The Design Lab is a workshop located within the first floor of the UGLi. This collaboration space houses tools, technical equiptment, and project consulting services for students, etc. The Awesomizer was created to create a centralized location in which we could create a collection of books that student's peers are reading -- books that are deemed awesome by the library's inhabitants and readers.

how was this project made?

Code for our project can be found on Github. For this project, we used a Raspberry Pi, a barcode scanner, and a computer. Our version of the project uses python for our barcode and data processing scripts, as well as SocketIO Flask for the server of this website. The scripts and code for this website can be found in our repository.
